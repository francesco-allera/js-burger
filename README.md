# EXERCISE 21 - BOOLEAN CAREERS TRAINEE:


### Languages used:

- HTML5
- CSS3
- JAVASCRIPT ES5


### Purpose of the exercise:

The user will be able to create his burger and know the price of his order.

He will necessarily assign a name to his burger.

Must select at least 2 ingredients.

If you insert a coupon code from those on our list, we apply a 20% discount on the total.

All this is generated (calculated) when the button is clicked.

The text where to insert the name of the hamburger is an input.
